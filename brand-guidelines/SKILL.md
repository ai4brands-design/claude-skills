# Anthropic Brand Styling

Automate the application of Anthropic's distinct typography and color palette.

## Official Palette

- **Carbon**: `#141413`
- **Sand**: `#faf9f5`
- **Stone**: `#e8e6dc`
- **Muted Stone**: `#b0aea5`
- **Accent - Clay**: `#d97757`
- **Accent - Sky**: `#6a9bcc`
- **Accent - Forest**: `#788c5d`

## Typography

- **Primary**: Inter (sans-serif)
- **Secondary**: System-ui, sans-serif

## Pro Tips

- Use "Sand" for main backgrounds and "Carbon" for primary text for a premium technical feel.
- Apply "Clay" or "Sky" sparingly for high-priority call-to-actions.
- Maintain consistent spacing based on an 8-point grid to align with the brand's precision.
