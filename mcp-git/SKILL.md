# Git MCP Skill

This skill integrates the Git MCP Server to provide version control capabilities.

## Capabilities

- Read repository history (logs)
- Search repository content (grep)
- Managing branches and tags
- Analyzing diffs

## Usage

Use this skill when the user asks to analyze a git repository, check history, or manage version control tasks.
