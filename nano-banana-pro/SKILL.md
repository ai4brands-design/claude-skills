# Nano Banana Pro

## Overview

This skill provides access to the "Nano Banana" collection of high-quality AI image prompts. It covers photorealistic portraits, 3D miniatures, product photography, and stylized aesthetics. Use this skill when the user wants to generate high-quality images or needs inspiration for visual prompts.

**Core Style**: High detail, natural lighting, specific camera settings (85mm, f/1.8).
