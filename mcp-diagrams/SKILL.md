# Diagrams MCP Skill

This skill generates diagrams.

## Capabilities

- Create Flowcharts
- Create Sequence Diagrams
- Create Class Diagrams
- Render using Mermaid syntax

## Usage

Use this when the user needs to visualize a process, logic, or system architecture.
