# Google Drive MCP Skill

This skill allows the agent to interact with Google Drive contents.

## Capabilities

- Search for files in Drive
- Read file content
- Upload or update files
- List folder contents

## Usage

Use this when the user mentions "Google Drive" or asks to read/write files to their cloud storage.
