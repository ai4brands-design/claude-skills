# ai4brands-design / claude-skills

This repository contains Claude skills for the **ai4brands-design** GitHub organization.

- GitHub org: https://github.com/ai4brands-design
- This repo: https://github.com/ai4brands-design/claude-skills

## What are skills?

Skills are modular AI "instruction sets" that extend Claude's capabilities for specific domains (e.g. SEO, Stripe, GitHub automation, frontend patterns, etc.). Each subdirectory is one skill.
