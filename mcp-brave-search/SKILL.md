# Brave Search MCP Skill

This skill provides internet search capabilities via Brave.

## Capabilities

- Search the web for queries
- Retrieve search snippets
- Local vs Global search

## Usage

Use this when the user asks to search the web and refers to "Brave" or when standard search tools are insufficient.
