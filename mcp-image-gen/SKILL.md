# Image Generation MCP Skill

This skill provides access to AI image generation models (like DALL-E, Stable Diffusion).

## Capabilities

- Generate images from text prompts
- Create variations of existing images

## Usage

Use this when the user asks to "generate an image of X" or "create a picture".
