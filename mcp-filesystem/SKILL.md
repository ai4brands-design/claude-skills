# Filesystem MCP Skill

This skill integrates the Filesystem MCP Server to provide enhanced file manipulation capabilities.

## Capabilities

- Read multiple files
- Write files
- List directories recursively
- Search for files

## Usage

Use this skill for complex file operations that go beyond standard tool capabilities, or when the user explicitly references "filesystem mcp".
