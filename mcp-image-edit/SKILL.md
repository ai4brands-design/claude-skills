# Image Editing MCP Skill

This skill allows basic image manipulation.

## Capabilities

- Resize and crop images
- Apply filters (blur, sharpen)
- Convert formats (PNG to JPG)
- Adjust brightness/contrast

## Usage

Use this when the user asks to resize, crop, or modify an existing image file.
